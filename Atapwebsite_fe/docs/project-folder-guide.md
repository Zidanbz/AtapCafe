# Project Folder Guide

```text
Atapwebsite_fe
```

Frontend Next.js untuk customer ordering dan admin panel.

```text
Atapwebsite_be
```

Backend TypeScript scaffold. Saat ini memiliki health endpoint dan siap dikembangkan menjadi API ordering.

```text
Atapwebsite_fe/src/app
```

Route Next.js App Router. File page sebaiknya tipis.

```text
Atapwebsite_fe/src/components
```

Komponen global lintas fitur, seperti `MobileShell` dan UI primitives.

```text
Atapwebsite_fe/src/features/ordering
```

Semua logic dan komponen customer ordering Atap.

```text
Atapwebsite_fe/src/features/admin
```

Semua logic dan komponen admin Atap.

```text
Atapwebsite_fe/public/images
```

Asset publik frontend, termasuk logo Atap dari folder desain.

```text
Atapwebsite_fe/docs/reference/Atap Desain
```

Referensi desain asli dari user.
